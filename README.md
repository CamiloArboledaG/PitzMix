# PitzMix

Videojuego online que consiste en el juego de pelota mesoamericano, donde se podrán manejar 3 dioses Aztecas y habrá que anotar puntos pasando el balón por un aro, cada dios tiene una habilidad única

#Tecnologías
- Unity
- C#
